# 涛斯拉 Webshell 管理工具 二开版

## 项目简介

本项目基于原版哥斯拉（Godzilla）Webshell 管理工具进行二次开发，核心优化了流量隐匿能力并新增多款免杀型 Webshell 载荷，解决原版在高安全防护环境下易被检测、拦截的问题，适用于安全测试、渗透测试授权场景。

> ⚠️ 重要声明：本工具仅用于**授权的安全测试、漏洞验证**场景，禁止用于任何非法攻击行为。使用本工具造成的一切法律责任，由使用者自行承担。

## 核心特性

### 1. 流量完全隐匿

- 移除原版固定特征的 HTTP 头、参数名、加密特征
- 自定义动态请求头（模拟浏览器真实请求）
- 流量加密方式升级：采用非对称加密 + 动态密钥协商，无固定加密特征
- 支持自定义 User-Agent、Referer、Cookie 等请求标识，模拟正常业务流量
- 支持分块传输、随机延时，规避流量特征检测

### 2. 新增免杀 Webshell 载荷

- 新增 PHP/Java/ASP.NET多语言免杀 Webshell，绕过主流 WAF、杀毒软件、EDR 检测
- 载荷采用内存加载、特征混淆、字符串动态拼接等免杀技术
- 支持自定义载荷模板，可快速适配不同防护环境

### 3. 兼容原版功能

- 完全保留原版哥斯拉的文件管理、命令执行、数据库管理、端口转发等核心功能
- 兼容原版已生成的 Webshell 连接（向下兼容）

## 环境要求

- 运行环境：JDK 1.8+（推荐 1.8.0_202）
- 支持操作系统：Windows/Linux/macOS
- 目标环境：PHP 5.4+ / Java 1.6+ / ASP.NET2.0+

## 快速使用

### 1. 工具启动

```
# Windows 启动
java -jar Godzilla_XST.jar

# Linux/macOS 启动
chmod +x Godzilla_XST.jar
java -jar Godzilla_XST.jar
```

### 2. 生成免杀 Webshell

1. 打开工具后，点击「生成」按钮
2. 选择目标语言（PHP/Java/ASP.NET）
3. 选择「免杀载荷」模板（推荐「通用免杀版」）
4. 设置连接密码、密钥（建议自定义复杂值）
5. 点击「生成」，保存生成的 Webshell 文件到目标服务器

### 3. 连接 Webshell

1. 点击「添加」按钮，填写目标信息：
   - 地址：目标 Webshell 访问 URL
   - 密码：生成 Webshell 时设置的密码
   - 脚本类型：对应生成的 Webshell 语言
   - 加密方式：选择「动态隐匿加密」
2. 点击「测试连接」，验证成功后点击「保存」
3. 双击连接即可进入管理面板

| :-----------: | :-------------------------: | :------------------: |

|  Java 免杀版  |   绕过 AWVS、AppScan、EDR   | SpringBoot/SSM 框架  |
| ASP.NET免杀版 | 绕过 IIS 防护、360 主机卫士 | .NET Framework 4.0+  |

## 流量隐匿配置（进阶）

1. 点击「设置」-「流量配置」
2. 自定义请求参数名（避免 `pass`、`cmd` 等敏感关键词）
3. 开启「动态请求头」：自动随机生成浏览器请求特征
4. 开启「流量混淆」：在请求体中插入随机无效数据，干扰特征检测

## 注意事项

1. 免杀载荷的有效性随安全软件规则更新而变化，建议定期更新载荷模板
2. 流量隐匿配置需与目标服务器环境匹配，避免因请求格式异常被拦截
3. 在高防护环境下，建议降低操作频率，避免因高频请求触发异常告警
